PrepWise is an AI-powered interview practice platform designed to help users prepare for technical and behavioral interviews more effectively.

The system dynamically generates interview questions based on selected topics or uploaded resumes, allowing users to simulate real interview scenarios. It evaluates user responses using AI, providing structured feedback, scores, and improvement suggestions.

Built with a modern full-stack architecture using React/Blazor and ASP.NET Core, PrepWise is fully containerized with Docker and supports scalable deployment with CI/CD pipelines.
